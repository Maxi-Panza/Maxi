Link Generator V1


Este es un programa que hize para la automatizacion de creacion de links para hosting ftp.. Aca explico como funciona

LOS ANTIVIRUS LO DETECTAN COMO VIRUS ASI QUE PROBABLEMENTE TENGAN QUE DESACTIVARLO
--------------------------
Esto hay maneras de arreglarlo pero no me voy a tomar el tiempo(tal vez algun dia)

1. Crear las carpetas en base a si es clientes, paises u otro
--------------------------
Primero tenemos que crear las carpetas que se van a usar para el directorio. Por ahora el programa no permite diferenciar entre la carpeta de clientes, paises, etc por lo cual tenemos que empezar a crear las carpetas. Ejemplo

```
paises/argentina/imagenes-whatapp/marcas/Fiat/Pulse/el-tipo-de-imagen/

Clientes/Peru/Domein/Zontes/V310/el-tipo-de-imagen/
```

2. Poner tanto el .EXE como el .PY en la carpeta padre
--------------------------
Tenemos que descomprimir el .rar que se descarga y habra dos archivos, un .exe y un .py 

2. Asegurarse que no contenga espacios ninguna carpeta o archivo
--------------------------
Actualmente no reemplaza los espacios, por lo cual cualquier espacio que haya no terminara funcionando el link


3. Asegurarse que siga el orden correcto
--------------------------
Actualmente no acepta otra indicacion mas que contenga las carpetas de imagen-interactiva, foto-agencia, colores, ficha-tecnica

4. Como usar el armador
--------------------------
Para usar el armador es necesario que hagan click derecho en el .py y seleccionen "Abrir con" y seleccionen el link_generator.exe 

Luego hacer click en el .py y se ejecutara el armador que creara un archivo en el escritorio con todos los links generados
